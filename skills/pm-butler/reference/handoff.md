# /pm-butler handoff

Structured transition between SDLC phases.

## Flow

1. Confirm outgoing persona's deliverable is complete (or log gaps as concerns).
2. Summarize context for incoming persona in a handoff block:
   - Phase completed
   - Key decisions
   - Open concerns
   - Explicit ask for next phase
3. Switch voice to incoming persona and continue.

## Chain

Oracle ? Sculptor ? Architect ? Alchemist ? Gatekeeper ? Shipper ? Librarian ? Oracle
